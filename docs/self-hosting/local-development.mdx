This guide walks you through running Agent Kanban locally for development and testing.

## Prerequisites

Before you begin, ensure you have:

- **Rust** (stable toolchain via [rustup](https://rustup.rs/))
- **Node.js 20+** and **pnpm** (`npm install -g pnpm`)
- **cargo-watch** for backend hot-reloading (`cargo install cargo-watch`)

## Step 1: Clone the Repository

```bash
git clone https://github.com/BloopAI/vibe-kanban.git
cd vibe-kanban
```

## Step 2: Install Dependencies

```bash
pnpm i
```

## Step 3: Run in Development Mode

A single command starts both the backend server and the frontend dev server with port auto-assignment:

```bash
pnpm run dev
```

This runs:
- **Backend** (`cargo watch`): Rust API server with hot-reload on file changes
- **Frontend** (Vite): React web app at the auto-assigned port

The terminal output will show the URLs for both services. Open the frontend URL in your browser.

## Running Backend and Frontend Separately

You can also run each part independently:

```bash
# Backend only (with hot-reload)
pnpm run backend:dev:watch

# Frontend only (Vite dev server)
pnpm run local-web:dev
```

Set ports manually via environment variables if needed:

```bash
FRONTEND_PORT=3000 BACKEND_PORT=8080 pnpm run dev
```

## Environment Configuration

Create a `.env` file at the workspace root for local overrides:

```env .env
# Override the backend port (default: auto-assigned)
BACKEND_PORT=8080

# Override the frontend port (default: auto-assigned)
FRONTEND_PORT=3000
```

No OAuth credentials, database setup, or external services are required. Agent Kanban stores all data in a local SQLite database.

## Type Checking

Run TypeScript and Rust type checks together:

```bash
pnpm run check
```

Or run them separately:

```bash
# Frontend TypeScript
pnpm --filter @vibe/local-web run check
pnpm --filter @vibe/web-core run check

# Backend Rust
cargo check --workspace
```

## Testing

```bash
# Rust unit and integration tests
cargo test --workspace

# Frontend linting
pnpm run lint
```

## Formatting

```bash
# Format all code (Rust + TypeScript)
pnpm run format
```

## Database

Agent Kanban uses SQLite via SQLx. Migrations run automatically on server startup. If you need to regenerate the SQLx offline query cache after changing queries:

```bash
pnpm run prepare-db
```

## Generating TypeScript Types

Shared types are generated from Rust structs using `ts-rs`. After changing any Rust structs annotated with `#[derive(TS)]`:

```bash
pnpm run generate-types
```

Do not edit `shared/types.ts` directly — it is auto-generated.

## Building the NPX Package

To test the distributable CLI package locally:

```bash
pnpm run build:npx
cd npx-cli
pnpm pack
```

Then test with:

```bash
npx ./npx-cli/vibe-kanban-<version>.tgz
```
